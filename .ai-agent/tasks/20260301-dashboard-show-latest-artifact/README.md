# Dashboard: 最新アーティファクト表示

## 目的・ゴール

ダッシュボード（`/`）を開いた際に、アーティファクト一覧ではなく最新のアーティファクトを直接表示する。
一覧はヘッダーのナビゲーションからアクセスできるようにする。

## 実装方針

### 現状

- `/` でダッシュボード（一覧テーブル）を表示
- `create` SSE イベントで新規ページに自動遷移
- `update`/`delete` でリロード

### 変更内容

ダッシュボードを「シェル」構造に変更:

1. **ヘッダー**: タイトル + ページセレクター（ドロップダウン/リスト）
2. **メインエリア**: 最新アーティファクトを iframe で表示
3. **空状態**: アーティファクトがない場合は従来の空メッセージを表示

iframe を使うことで、ページのスタイルがダッシュボードのヘッダーと干渉しない。

### SSE 連携

- `create`: ヘッダーのリストに追加 + 自動的にそのページに切り替え
- `update`: ヘッダーのリスト情報を更新（タイトル・名前変更対応）
- `delete`: ヘッダーのリストから削除 + 別のページに切り替え
