# プロダクトビジョン

## ミッション

AI アシスタントが生成した HTML コンテンツを、MCP プロトコル経由で即座にブラウザプレビューできる環境を提供する。

## ターゲットユーザー

- Claude Code や MCP 対応 AI ツールを使用する開発者
- AI とのディスカッションや調査で、リッチな可視化（図表・比較表・チャート）を活用したいユーザー

## ユースケース

1. **HTML プレビュー**: AI が生成した HTML/CSS/JS コードのレンダリング結果を即座にブラウザで確認
2. **ディスカッション可視化**: アイデア整理時に論点構造・選択肢比較表・フロー図を HTML で生成・表示
3. **技術調査の可視化**: 調査結果をチャートや比較表で可視化
4. **タスク計画の可視化**: タスク依存関係図やタイムラインを HTML で表示

## 主要機能

### 提供中（v1.0.0）

- `create_page`: HTML ページを作成し、ブラウザで閲覧可能な URL を返す
- `update_page`: 既存ページの HTML コンテンツ・タイトルを更新（ホットリロード対応）
- `destroy_page`: 不要になったページを削除（削除済み履歴として保持）
- `get_pages`: 作成済みページの一覧を取得（`includeDeleted` で削除済みも表示可能）
- `get_page`: 特定ページの HTML コンテンツを取得
- `add_scripts`: CDN スクリプト（Mermaid.js、Chart.js 等）をページに追加
- `add_stylesheets`: 外部スタイルシートをページに追加
- ダッシュボードページ（アーティファクト一覧表示、各ページへの遷移、削除済み履歴表示）
- ブラウザ自動オープン（初回 `create_page` 時にダッシュボードを自動で開く、`--no-open` で無効化）
- 新規アーティファクト作成時の自動切り替え（SSE 経由でダッシュボードから新ページへ自動遷移）
- 動的ポート割り当てによる複数セッション同時利用
- 履歴管理（`--max-pages` で保持上限、`--ttl` で有効期限を設定可能）
